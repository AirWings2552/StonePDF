Why it matters

PDF 里的内部链接不应该打断你的思考

1

看到引用

教材或论文常常写着 “see Eq. 2.14” 或 “as shown in Fig. 3”。你正在读第 7 章，不想跳回第 2 章。

2

悬浮预览

把鼠标停在内部链接上，StonePDF 会弹出目标页局部预览，直接看到公式、图表或定义。

3

继续阅读

确认引用内容后，鼠标移开即可回到原文。需要认真对照时，也可以固定预览窗并拖动调整大小。

Install

安装步骤

  1. 下载安装包 打开下载页，选择最新版 Windows installer。
  2. 运行安装 如果 Windows 提示未知发布者，确认来源是 StonePDF 项目后继续安装。
  3. 打开 PDF 打开教材、论文或技术文档，把鼠标悬浮在内部链接上试试预览窗。

开源项目
